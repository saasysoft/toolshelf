### Cursor

Cursor is a fork of VS Code with deeply integrated AI capabilities. It provides AI-powered code completion, multi-file editing with Composer, inline chat, and codebase-aware context. One of the most polished AI coding experiences available.

[Website](https://cursor.com) | [GitHub](https://github.com/getcursor/cursor)

### Zed

Zed is a next-generation code editor built from scratch in Rust for maximum performance. It features real-time collaboration, integrated AI assistance, Tree-sitter for syntax highlighting, and GPU-accelerated rendering. From the team behind Atom and Electron.

[Website](https://zed.dev) | [GitHub](https://github.com/zed-industries/zed)

### Neovim

Neovim is a modern refactor of Vim with built-in LSP client, Tree-sitter integration, Lua scripting, and an async plugin architecture. It is the foundation for a massive ecosystem of plugins and distributions (LazyVim, AstroNvim, NvChad) that create full IDE experiences.

[Website](https://neovim.io) | [GitHub](https://github.com/neovim/neovim)
