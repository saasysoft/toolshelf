Looking for the best media and design tools? We analyzed 15 tools in this category using ToolShelf's quality scoring system — factoring in maintenance activity, community adoption, documentation quality, and more.

Here are the top picks for 2026.

### 1. n8n (175.2k stars)

n8n is a fair-code workflow automation platform with 400+ integrations and a visual node-based editor. It supports complex branching logic, custom JavaScript/Python code nodes, webhooks, and can be self-hosted. Bridges the gap between no-code and full-code automation.

**Pricing:** freemium — [Website](https://n8n.io)

### 2. lazygit (72.6k stars)

lazygit provides a full-featured terminal user interface for Git. Stage files, resolve merge conflicts, cherry-pick commits, rebase interactively, and browse diffs all with keyboard shortcuts. Makes complex Git operations visual and approachable.

**Pricing:** open-source — [Website](https://github.com/jesseduffield/lazygit)

### 3. Kitty (31.3k stars)

Kitty is a GPU-accelerated terminal emulator that uses OpenGL for rendering. It supports image display (via its graphics protocol), ligatures, tabs, splits, remote control via IPC, and extensibility through kittens (small Python programs). Highly customizable and performant.

**Pricing:** open-source — [Website](https://sw.kovidgoyal.net/kitty/)

### 4. DevToys (31.0k stars)

DevToys bundles 30+ offline developer utilities into a single app: JSON formatter, JWT decoder, Base64 encoder, UUID generator, hash calculator, regex tester, color picker, and more. Works completely offline with a clean native interface.

**Pricing:** open-source — [Website](https://devtoys.app)

### 5. WezTerm (24.3k stars)

WezTerm is a GPU-accelerated cross-platform terminal emulator and multiplexer written in Rust. It is configured entirely in Lua, supports ligatures, image rendering, multiplexing (like tmux built-in), and has excellent cross-platform consistency.

**Pricing:** open-source — [Website](https://wezfurlong.org/wezterm/)

### 6. Feishin (7.1k stars)

A modern self-hosted music player. Built with Typescript. Open-source project with 7.1k GitHub stars.

**Pricing:** open-source — [Website](https://github.com/jeffvli/feishin)

**Pricing:** open-source — [Website](https://github.com/maxdorninger/MediaManager)
